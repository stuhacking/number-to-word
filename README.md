Number 2 Word
=============

A Clojure library that converts numbers given in arabic numerals into readable
English text.

## Usage

Interactive REPL or run from command line:

Example usage:

    $ java -jar num2word.jar 10 20 400 10842
    ten
    twenty
    four hundred
    ten thousand, eight hundred and forty two
